# 排序
|排序方法 |稳定性 |最好复杂度 |最坏复杂度 |期望复杂度 |
| ---------- | :-----------:  | :-----------: | :-----------: | :-----------: |
|冒泡排序|稳定|O(n)|O(n2)|O(n2)|
|选择排序|稳定|O(n2)|O(n2)|O(n2)|
|插入排序|稳定|O(n)|O(n2)|O(n2)|
|快速排序|不稳定|O(n)|O(n2)|O(nlogn)|
|归并排序|稳定|O(nlogn)|O(nlogn)|O(nlogn)|
